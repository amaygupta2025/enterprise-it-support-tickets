## Incident Title
User unable to connect via RDP (Remote Desktop)

## Environment
- Windows 10(client)
- Windows Server (Remote system)
- Enterprise corporate network
- RDP(Port 3389)

## Issue Description
User reported being unable to connect to remote system using Remote Desktop Connection. Error message indicated that the remote computer could not be reached.

## Impact 
User was unable to access the remote system required for daily work, resulting in work stoppage until the issue was resolved.
